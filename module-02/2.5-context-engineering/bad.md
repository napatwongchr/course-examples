❌ Prompt Engineering Layer เดียว — ไม่มี Context Layer

```
คุณเป็น product manager ผู้เชี่ยวชาญ EdTech
ช่วยแนะนำ feature ที่ควรมีใน learning app สำหรับนักศึกษา
ตอบเป็น bullet points 5 ข้อ ภาษาไทย
```

---

> 💬 ส่งก่อน แล้วสังเกตว่าคำตอบที่ได้ generic มาก
> เช่น Notification, Progress Tracking, Gamification...
> เพราะ AI ไม่รู้อะไรเกี่ยวกับ project ของเราเลย
> ทั้งที่ Prompt มีโครงสร้างดีมาก มี role + format + action ครบ
