ฉันมี Prompt สำหรับสร้าง PRD ของระบบ Authentication
แต่ PRD ที่ได้ออกมา generic เกินไป และไม่ตรงกับสิ่งที่ต้องการ

ช่วยวิเคราะห์ Prompt นี้โดยใช้ 4 ปัจจัยต่อไปนี้
สำหรับแต่ละปัจจัย ให้ระบุว่า:
(1) มีปัญหาด้านนี้ไหม
(2) ปัญหาคืออะไรโดยเฉพาะ
(3) แนะนำวิธีแก้ไข

ปัจจัยที่ 1 — คำสั่งไม่ชัดเจน
คำสั่งกำกวม ไม่ระบุรูปแบบ หัวข้อ หรือความลึกของผลลัพธ์

ปัจจัยที่ 2 — ข้อมูลไม่เพียงพอ
ขาดข้อมูลสำคัญที่ AI ต้องใช้เพื่อสร้างผลลัพธ์ที่ถูกต้องและ specific

ปัจจัยที่ 3 — ข้อมูลเกินจำเป็น
มีข้อมูลที่ไม่เกี่ยวข้องกับงานที่ขอ ทำให้ AI โฟกัสผิดจุด

ปัจจัยที่ 4 — บริบทไม่เหมาะสม
ไม่ได้กำหนดบทบาทของ AI หรือบริบทที่ชัดเจนว่า output นี้จะใช้ทำอะไร

## Prompt ที่จะวิเคราะห์:

เราเป็น Startup ที่กำลังพัฒนา Platform จัดการธุรกิจ SME ครบวงจร
มี feature หลักๆ คือ: Dashboard, Inventory Management, Sales Report,
CRM, HR Management, Payroll, Document Management, Notification System
รองรับ Multi-tenant architecture และมี Mobile App ด้วย

ตอนนี้อยากพัฒนาระบบ Authentication ใหม่
ช่วยสร้าง PRD ให้หน่อยครับ
