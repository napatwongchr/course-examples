# ✅ Context จัดดี — แยกหมวดชัดเจน มีข้อมูลครบ

เปิด chat ใหม่ แล้วส่ง prompt นี้:

```
## แอปที่พัฒนา
Personal Learning Assistant (PLA) — เว็บแอปช่วยนักศึกษาไทยทบทวนเนื้อหาที่เรียน

## ผู้ใช้หลัก
นักศึกษามหาวิทยาลัย อายุ 18–24 ปี ใช้ผ่านเว็บเบราว์เซอร์

## Feature ที่มีอยู่แล้ว
- บันทึก Note แบ่งตาม Topic
- Timer จับเวลาการเรียน

## ข้อจำกัดทีม
- Developer 2 คน
- เหลือเวลา 6 สัปดาห์
- ยังอยู่ช่วง Prototype — ยิ่งไม่ต้องการ backend ซับซ้อนยิ่งดี

แนะนำ feature ถัดไปที่ควรพัฒนา พร้อมเหตุผลที่สอดคล้องกับข้อจำกัดของทีม
```

---

> 💬 สังเกตเลยครับ — ตอนนี้ AI รู้ว่าแอปคืออะไร ผู้ใช้คือใคร
> feature ที่มีแล้วคืออะไร และทีมมี constraint อะไร
> คำตอบที่ได้จะเจาะจงและต่อยอดจากสิ่งที่มีอยู่ได้จริง
> ไม่ใช่แค่แนะนำ feature ทั่วๆ ไปที่ทุกแอปก็มี
> นั่นคือความต่างของ Context ที่จัดดีครับ
