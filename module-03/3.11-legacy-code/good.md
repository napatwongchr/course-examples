# ✅ Prompt อธิบายโครงสร้าง Codebase — สำหรับ Onboard Developer ใหม่

```
คุณเป็น senior developer ที่กำลัง onboard developer ใหม่เข้าร่วมทีม PLA

จาก codebase ที่แนบมานี้ ช่วยอธิบาย:

1. ภาพรวมของโปรเจกต์ — ทำอะไร ใช้ tech stack อะไร
2. โครงสร้าง folder หลักและหน้าที่ของแต่ละ folder
3. data flow หลัก — ข้อมูลไหลจากไหนไปไหน
4. ฟังก์ชัน/component ที่สำคัญที่สุด 3-5 ตัวที่ต้องเข้าใจก่อน

[แนบ file tree และ code หลัก]

ตอบในรูปแบบที่ developer ที่ไม่รู้จักโปรเจกต์เลยอ่านแล้วเข้าใจได้ภายใน 5 นาที
```

---

> 💬 ดีมากครับ สังเกตสิ่งที่ prompt นี้ทำ:
>
> — บอก **role** ให้ AI ชัดเจน (senior developer ที่กำลัง onboard)
> — บอก **audience** ที่แท้จริง (developer ใหม่ที่ไม่รู้จักโปรเจกต์เลย)
> — จำกัด **scope** ด้วยตัวเลข (3-5 ตัว) ไม่ให้ตอบกว้างเกินไป
> — กำหนด **success criteria** ชัดเจน (อ่านแล้วเข้าใจได้ภายใน 5 นาที)
>
> พอ AI รู้ว่า audience คือใคร คำตอบจะตรงจุดและ actionable กว่ามากครับ
