# Vibe Coding Demo — Pomodoro Timer

> 💡 Demo นี้แสดงให้เห็นว่า Vibe Coding คือการ "อธิบาย" สิ่งที่ต้องการ
> ไม่ใช่การ "เขียน" code ทีละบรรทัด — ส่ง prompt นี้ใน Claude Code แล้วดูผลลัพธ์

---

สร้าง Pomodoro Timer เป็น single HTML file
- วงกลม countdown แสดงเวลาที่เหลือ 25 นาที
- ปุ่ม Start / Pause / Reset
- เมื่อหมดเวลาให้เปลี่ยนสีและมีเสียงแจ้งเตือน
- หน้าตาดูดี สีโทน dark
