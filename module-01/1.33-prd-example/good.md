# ✅ PRD Prompt — ระบุโครงสร้างครบ 6 ส่วน

สร้าง PRD สำหรับแอป "Study Buddy" — เว็บแอปสำหรับนักศึกษา
มหาวิทยาลัยที่ช่วยจับคู่กลุ่มเรียนตามวิชาและเวลาที่ว่างตรงกัน

PRD ต้องมีครบ 6 ส่วน:
1. Executive Summary
2. Problem Statement
3. Product Goals & Success Metrics (ระบุตัวเลขที่วัดได้)
4. User Stories อย่างน้อย 3 ข้อ (รูปแบบ As a... I want... So that...)
5. Functional Requirements
6. Timeline & Milestones (2 Phase)

เขียนให้ครบถ้วนและใช้งานได้จริงในทีม Developer
