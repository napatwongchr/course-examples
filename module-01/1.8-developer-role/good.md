# ✅ Snake Game — Prompt ระบุ Requirement ชัดเจน

คุณเป็น Frontend Developer ผู้เชี่ยวชาญ HTML, CSS และ JavaScript

งาน: สร้างเกมส์งู (Snake Game) แบบ Single HTML File ที่พร้อมเล่นได้ในเบราว์เซอร์ทันที

Requirements:
- ไฟล์เดียว: HTML + CSS + JS รวมอยู่ในไฟล์ .html ไฟล์เดียว ไม่มีไฟล์แยก
- Canvas ขนาด 400x400px พื้นหลังสีเข้ม (#1a1a2e)
- งูสีเขียวสว่าง (#00ff88) และอาหารสีแดง (#ff4757)
- ควบคุมด้วย Arrow Keys หรือ WASD
- แสดง Score มุมบนขวา และเก็บ High Score ไว้ใน localStorage
- เมื่อชนตัวเองหรือขอบ ให้แสดง Game Over พร้อม Score และปุ่ม Restart
- ความเร็วเพิ่มขึ้นเล็กน้อยทุก 5 คะแนน

Output: ไฟล์ HTML เดียว พร้อม code ที่ clean และมี comment อธิบายแต่ละส่วน
