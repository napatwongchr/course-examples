## Scene 2 — Demo

**เป้าหมาย:** ให้ผู้ชมเห็นว่า AI ช่วยอ่าน Coverage Report และบอกได้เลยว่า "ต้องเพิ่ม Test ตรงไหน" โดยไม่ต้องตีความเอง

| # | ทำบนหน้าจอ | พูด |
|---|-----------|-----|
| 1 | เปิด Claude.ai → New conversation | — |
| 2 | **แสดง Coverage Report** จากไฟล์ `01-coverage-report-input.md` | "นี่คือ Coverage Report ที่ได้จากการรัน npm run test -- --coverage ครับ" |
| 3 | Copy prompt จาก `02-coverage-analysis-prompt.md` พร้อมเนื้อหา Report → Paste → Send | "ให้ AI ช่วยแปลผลและบอกว่าต้องทำอะไรต่อครับ" |
| 4 | รอ response | — |
| 5 | **ชี้ที่ "Coverage สรุปภาพรวม"** | "AI สรุปภาพรวมให้เลย — Coverage รวมอยู่ที่เท่าไหร่" |
| 6 | **ชี้ที่ "ไฟล์ที่ Coverage ต่ำที่สุด"** | "และชี้ได้เลยว่าไฟล์ไหนที่ Coverage ต่ำที่สุด ต้องแก้ก่อน" |
| 7 | **ชี้ที่ "บรรทัดที่ยังไม่ถูก Test"** | "Uncovered lines — บรรทัดที่ยังไม่มี Test ผ่านเลย" |
| 8 | **ชี้ที่ "คำแนะนำ Priority"** | "AI ยังบอก Priority ว่าควรเขียน Test ไฟล์ไหนก่อน" |
| 9 | หยุดนิ่ง 2 วินาที | "จาก Report ที่อ่านยาก AI แปลงมาเป็น Action List ที่ทำตามได้เลยครับ" |
