## 💡 จุดเน้นที่น่าสนใจสำหรับผู้ชม

- **ไฟล์ที่เชื่อมกัน:** `store/mock-store.ts` → `services/study-session.service.ts` + `services/topic.service.ts` → API routes — เป็น chain ที่ดูเหมือน share state แต่จริงๆ ไม่ใช่
- **Silent bug:** `createStudySession()` ใน `mock-store.ts` มี `if (topic) { ... }` — ถ้า topic ไม่พบ ระบบ return 201 Created แต่ไม่อัปเดตอะไรเลย ไม่มี error ใดโยน
- **การแก้ที่ผ่านมาล้มเหลวเพราะอะไร:** เพิ่ม `await`, `refetchOnWindowFocus` — ทั้งหมดแก้ที่ฝั่ง client แต่ปัญหาอยู่ที่ฝั่ง server's data layer
